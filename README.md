# react_redux_cart
React + Redux 做的购物车案例

### React表单中受控组件与非受控组件

> 受控组件与非受控组件的区别

```
# 受控组件
  受控组件的状态由React组件控制
	
# 非受控组件
  非受控组件的状态由DOM控制
```

> 非受控组件Ref使用的两种途径

```
# 通过refs获取dom
  1、给表单元素添加ref属性
  2、通过 this.refs.xxx 拿到dom元素
  
# React.createRef()方式
  参考:https://react.docschina.org/docs/refs-and-the-dom.html
```
